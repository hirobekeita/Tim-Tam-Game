# Tim-Tam-Game
ティムタムゲーム

## ゲームの説明
「ティム」と「タム」がランダムに13回表示されます（7回表示後に改行して6回表示）。14回目に何がくるかを当てるゲームです。

**注意**: 正解は必ず回答の逆になります！
- 「ティム」と答えた場合 → 正解は「タム」
- 「タム」と答えた場合 → 正解は「ティム」

## 実行方法

### GUI版（推奨）
```bash
python3 timtam_game_gui.py
```

### CLI版
```bash
python3 timtam_game.py
```

## 必要環境
- Python 3.x
- tkinter (GUI版の実行に必要、通常はPythonに標準で含まれています)
